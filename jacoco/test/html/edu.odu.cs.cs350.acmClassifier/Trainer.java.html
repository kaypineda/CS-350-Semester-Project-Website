/**
 * Class used to retrain/ fine tune ML network
 * @author Team 7
 */

public class Trainer
{
  /*
  Is a class for the trainers for our neural networks. Given a set of documents, as well as the correct classifications for each document, it's going to
  ask each of a set of neural networks to determine the classification for each document, then grade them based on how well they did (the grade is known
  as the "fitness function"). Each generation, it'll grade a set of better and better neural networks, thus it can be expected that they will have higher
  and higher fitness scores. We can stop once the neural networks achieve an accuracy of about 80%.
  */
}
